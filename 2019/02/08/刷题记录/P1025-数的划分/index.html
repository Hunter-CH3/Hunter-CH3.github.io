题目

传送门|P1025 数的划分

DFS

DP

设$f[i, j]$表示将$i$分成$j$组的方案数。

状态转移方程为：$f[i, j] = f[i - 1, j - 1] + f[i - j, j]$

初始状态：$f[i, 1] = 1$

解释：

由于不考虑顺序，因此不妨将分组情况按升序排序。

方案分成两种：

  1. 首位为1。对于这一类分组，其方案数等于$f[i-1, j-1]$。
  2. 首位不为1。对于这一类分组，如果从每一组中拿走1，不会改变方案数；此时方案数等于$f[i-j, j]$。
