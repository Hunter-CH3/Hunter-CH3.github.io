题目

[洛谷|P1028|数的计算][ https://www.luogu.org/problemnew/show/P1028]

思路

递归

直接上代码

递推

考虑到有数学关系

f(x) = f(1) + f(2) + … + f(x / 2)

从而有

x % 2 == 0:

f(x) = f(x - 1) + f(x / 2)

x % 2 == 1:

f(x) = f(x - 1);

因此可以开一个数组，从1到n进行遍历，利用递推得到结果

递推的思想中包含了“记忆化搜索”，即建一个数表用来存储算过的值，如果这个值算过就直接调用不再计算，常见的代码行为
